Rip’n’Listen Demo

This demo ripper can save ripped data into several formats and provides the option to listen to the audio as it is being ripped.  This of course slows processing down to 1x.  For all the different formats, except wav, you will need Third Party Libraries.  See the Introduction for more information.  If you don’t have some encoder for some particular format on your system, remove the corresponding output component and all references to it from the project.

This demo requires CDRip.dll.
