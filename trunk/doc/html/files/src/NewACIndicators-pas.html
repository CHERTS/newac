TGainIndicator

Calculates the perceived gain (loudness).  Descends from TAuConverter.  This component provides you the perceived gain for the audio data passing through.  See the DirectSoundRecorder demo for an example.  The component requires libgain.dll.

Properties

GainValue

Returns the curreent gain value.  This value is calculated in abstract units in logarithmic scale and varies from ~ -1 (silence) to ~ 60 (the maximum loudness).

Interval

Use this property to set the interval (in milliseconds) between two OnGainData events.

OnGainData

OnGainData event is called periodically with the period of approximately the Interval milliseconds.  You can use this event to update your GUI gain indicators with the current GainValue.
